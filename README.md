# Analisis-Pola-Pesan-Terkait-Bencana-Alam-di-Media-Sosial

Proyek ini bertujuan mengeksplorasi dan mengekstrak frequent patterns dari pesan-pesan terkait bencana alam yang tersebar di media sosial. Fokus analisis diarahkan pada hubungan antara kata-kata atau frasa dalam pesan dengan kategori bantuan yang relevanâ€”seperti makanan, medis, evakuasi, perlindungan, dan lainnya.

Dengan memanfaatkan algoritma Frequent Pattern Mining (khususnya FP-Growth dan Apriori), proyek ini berupaya menemukan asosiasi yang kuat antar kata, sehingga pola komunikasi publik selama bencana bisa dipetakan dengan lebih akurat dan bermakna.
